srlib_js

Overview

The Sketch Recognition Library Javascript module (srlib_js) is a set of Javascript classes and methods for working with the Sketch Recognition Library made available through the Sketch Recognition Lab at Texas A&M University.

Installation

To setup and use the tool, simply run the following from the command line:

npm install

or, if you need the developer dependencies:

npm install --dev

Node's package manager, npm, will automatically handle installing of the dependencies needed for building documentation and running tests.

Available NPM Commands

Building the Documentation

To update the documentation, run npm run doc.

Running Tests

Run tests with npm run test. Alternatively, use the browser implementation by opening the "test.html" file inside the "test" directory in your browser.
